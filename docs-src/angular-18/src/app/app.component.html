Material-inspired multiselect, without breaking the classic contract.

This release line promotes <angular-multiselect> as the primary selector, while still accepting the legacy <angular2-multiselect> API, ngModel binding, event outputs, and custom template hooks while refining the UI to feel much closer to Angular Material surfaces and field patterns.

{{ pill }}
API stability first Inputs, outputs, settings keys, and template slots stay aligned with the code people already ship.
Material-like visual language Surface elevation, rounded outlines, calmer color tokens, and cleaner dropdown states.
Modern line foundation This shell powers the remodulated lines from Angular 14 through Angular 21 while keeping the legacy selector contract intact.
