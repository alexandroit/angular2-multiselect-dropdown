Quick API reference

The maintained line keeps the familiar API while documenting the most commonly used inputs and outputs in one place.

Core inputs

[data]Array of available dropdown items.
[(ngModel)]Two-way bind the selected items list.
[settings]Pass text, search, grouping, and lazy loading configuration.
disabledDisable interaction when the parent form requires it.

Templates

<c-item>Provide a custom item template for the dropdown list.
<c-badge>Customize the selected badge rendering.
tagToBodyRender the menu outside clipping containers when needed.

Outputs

(onSelect)Emits the selected item.
(onDeSelect)Emits when a selected item is removed.
(onSelectAll)Emits when all visible items are selected.
(onDeSelectAll)Emits when all selections are cleared.
